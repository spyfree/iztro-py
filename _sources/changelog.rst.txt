更新日志
========

v0.3.3 (2025-01-18)
-------------------

**类型系统修复**

* 修复 Python 3.8-3.9 语法兼容性问题
* 将 ``|`` 联合类型语法改为 ``Union[]``
* 迁移 Pydantic v2 ``ConfigDict``
* 修复 35+ 个 mypy 类型标注错误

**CI/CD**

* 添加 GitHub Actions CI workflow
* 添加自动化测试和类型检查
* 添加文档自动构建和部署
* 添加 PyPI 发布 workflow

**文档**

* 添加完整的 Sphinx 文档
* 添加快速开始指南
* 添加 API 参考文档
* 添加使用示例
